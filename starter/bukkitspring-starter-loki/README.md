# BukkitSpring Loki Starter

Platform-agnostic Loki log shipping for BukkitSpring.

## Docs
- English: `USAGE_EN.md`
- Chinese: `USAGE_CN.md`

## Install
1. Copy `bukkitspring-starter-loki-*.jar` to `plugins/BukkitSpring/starters/`.
2. Restart the server.

## Config
Edit `plugins/BukkitSpring/config.yml` (section `loki`).

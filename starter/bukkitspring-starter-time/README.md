# BukkitSpring Time Starter

Time utilities starter for BukkitSpring, including:
- Unified current-time access with configurable zone and debug offset
- Optional system-time adjustment (for local debugging only)

## Docs
- English: `USAGE_EN.md`
- Chinese: `USAGE_CN.md`

## Install
1. Copy `bukkitspring-starter-time-*.jar` to `plugins/BukkitSpring/starters/`.
2. Restart the server.

## Config
Edit `plugins/BukkitSpring/config.yml` (section `time`).
